# TubeFlow Frontend (Vercel Ready)

## Local Development
```bash
npm install
npm run dev
```

## Build & Preview
```bash
npm run build
npm run preview
```

## Vercel Deployment
1. Install Vercel CLI: `npm i -g vercel`
2. From root: `vercel --prod`
3. Set env var: `VITE_API_BASE_URL=https://your-backend.onrender.com`

Built with Vite + React + TailwindCSS.
